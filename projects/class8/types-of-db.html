Types of Database

There are two types of database:

  1. Flat File database: A flat database is a database designed around a single table. It stores and retrives information but do not link different tables together. The flat design puts all databse information in one table or list, with fields to represent all parameters. For example: MS Excel.
  2. Relational database: A relational database stores the data in serveral tables and link those tables together to get a common piece of information. It links tables together, so that different information from different tables can be accessed easily. It also reduces duplicity of data. For example: Microsoft Access, Microsoft SQL, Oracle.
